# API Playground

The `<APIPlayground>` component embeds a fully interactive HTTP client inside your documentation. Readers can test API endpoints with real requests without leaving the page.

## Live Demo

Try it now — this makes a real HTTP request to a public API:

<APIPlayground
  method="GET"
  url="https://jsonplaceholder.typicode.com/users"
  title="List Users"
  description="Fetches a list of all users from the JSONPlaceholder test API."
  parameters={[
    { name: "_limit", type: "number", required: false, description: "Limit the number of results.", defaultValue: "10" }
  ]}
/>

## POST Request with Body

<APIPlayground
  method="POST"
  url="https://jsonplaceholder.typicode.com/posts"
  title="Create Post"
  description="Creates a new post. JSONPlaceholder simulates the creation and returns the new object."
  headers={{ "Content-Type": "application/json" }}
  body={{ "title": "My Post", "body": "Post content here.", "userId": 1 }}
/>

## Authenticated Endpoint

<APIPlayground
  method="GET"
  url="https://jsonplaceholder.typicode.com/users/1"
  title="Get User (with Auth)"
  description="Demonstrates Bearer token authentication. The token is sent in the Authorization header."
  authType="bearer"
  servers={[
    { url: "https://jsonplaceholder.typicode.com", description: "Production" },
    { url: "https://staging.jsonplaceholder.typicode.com", description: "Staging" }
  ]}
/>

## Usage in MDX

```mdx
<APIPlayground
  method="GET"
  url="https://api.example.com/users"
  title="List Users"
  description="Returns a paginated list of users."
  parameters={[
    {
      name: "page",
      type: "number",
      required: false,
      description: "Page number (1-indexed).",
      defaultValue: "1"
    },
    {
      name: "limit",
      type: "number",
      required: false,
      description: "Items per page.",
      defaultValue: "20"
    }
  ]}
  headers={{
    "Accept": "application/json"
  }}
  authType="bearer"
/>
```

## Authentication Types

| `authType` | Description | Fields |
| --- | --- | --- |
| `none` | No authentication | — |
| `bearer` | Bearer token | Token field |
| `apiKey` | API key in header | Key name + value |
| `basic` | HTTP Basic auth | Username + password |

## Props Reference

| Prop | Type | Required | Description |
| --- | --- | --- | --- |
| `method` | `'GET' \| 'POST' \| 'PUT' \| 'DELETE'` | Yes | HTTP method. |
| `url` | `string` | Yes | Endpoint URL. |
| `title` | `string` | No | Display title. |
| `description` | `string` | No | Description shown above the playground. |
| `parameters` | `Parameter[]` | No | Pre-populated query parameters. |
| `headers` | `Record<string, string>` | No | Pre-populated request headers. |
| `body` | `object` | No | Pre-populated JSON request body. |
| `authType` | `'bearer' \| 'apiKey' \| 'basic' \| 'none'` | No | Authentication method. |
| `servers` | `Array<{ url, description }>` | No | Server environment options. |

### Parameter Object

| Field | Type | Description |
| --- | --- | --- |
| `name` | `string` | Parameter name. |
| `type` | `string` | Type hint (e.g. `"string"`, `"number"`). |
| `required` | `boolean` | Whether the parameter is required. |
| `description` | `string` | Parameter description. |
| `defaultValue` | `string` | Pre-populated value. |
