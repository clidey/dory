# Icons

The `<Icon>` component renders icons inline within text or as standalone visual elements. It supports **Lucide** icons by default and falls back to **FontAwesome** when a Lucide icon isn't found.

## Basic Usage

```mdx
<Icon icon="rocket" />
<Icon icon="code-2" />
<Icon icon="check-circle" />
```

**Output:**

<Icon icon="rocket" /> <Icon icon="code-2" /> <Icon icon="check-circle" />

## With Color

```mdx
<Icon icon="heart" color="#ef4444" />
<Icon icon="star" color="#f59e0b" />
<Icon icon="shield-check" color="#22c55e" />
```

**Output:**

<Icon icon="heart" color="#ef4444" /> <Icon icon="star" color="#f59e0b" /> <Icon icon="shield-check" color="#22c55e" />

## With Size

```mdx
<Icon icon="zap" size={16} />
<Icon icon="zap" size={24} />
<Icon icon="zap" size={40} />
```

**Output:**

<Icon icon="zap" size={16} /> <Icon icon="zap" size={24} /> <Icon icon="zap" size={40} />

## Lucide Icons in Context

Icons work well alongside text:

```mdx
<Icon icon="terminal" size={16} /> Run `dory build` to compile your site.

<Icon icon="package" size={16} /> Install with `npm install -g @clidey/dory`.

<Icon icon="git-branch" size={16} /> Push the `dist/` directory to your hosting provider.
```

**Output:**

<Icon icon="terminal" size={16} /> Run `dory build` to compile your site.

<Icon icon="package" size={16} /> Install with `npm install -g @clidey/dory`.

<Icon icon="git-branch" size={16} /> Push the `dist/` directory to your hosting provider.

## FontAwesome Fallback

When a Lucide icon isn't found, Dory falls back to FontAwesome. Specify the icon type with `iconType`:

```mdx
<Icon icon="github" iconType="brands" size={24} />
<Icon icon="twitter" iconType="brands" size={24} />
<Icon icon="npm" iconType="brands" size={24} />
```

**Output:**

<Icon icon="github" iconType="brands" size={24} /> <Icon icon="twitter" iconType="brands" size={24} /> <Icon icon="npm" iconType="brands" size={24} />

## Props Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `icon` | `string` | — | Icon name in kebab-case (e.g. `"check-circle"`). |
| `iconType` | `'solid' \| 'regular' \| 'brands'` | `'solid'` | FontAwesome icon type (used as fallback). |
| `color` | `string` | Inherits from theme | CSS color value (hex, rgb, named color). |
| `size` | `number` | `24` | Icon size in pixels. |
| `className` | `string` | — | Additional CSS classes. |
