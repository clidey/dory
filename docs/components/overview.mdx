## Using Components

All components are available in `.mdx` files without any imports:

```mdx
---
title: 'My Page'
---

# My Page

<Note>
  This is a note callout — no import needed.
</Note>

<CardGroup cols={2}>
  <Card title="Getting Started" icon="rocket" description="Set up your project." />
  <Card title="Configuration" icon="settings" description="Customize your site." />
</CardGroup>
```

<Note>
  All components are built into Dory. You never need to install or import anything — just use them directly in your pages.
</Note>
