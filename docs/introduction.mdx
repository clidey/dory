# Introduction

## Welcome to Docucod and Dory

This documentation is powered by two complementary tools:

1. **Docucod** - An AI-powered documentation assistant that automatically generates and maintains documentation
2. **Dory** - A lightweight static site generator specifically designed to deploy dory-generated documentation

## What is Docucod?

Docucod analyzes your codebase and automatically:
- Generates comprehensive documentation
- Updates existing documentation when code changes
- Maintains consistent style and formatting
- Ensures accuracy and completeness
