# Theming

Dory provides a comprehensive theming system that lets you match your docs to your product's visual identity. You can control brand colors, background/surface colors, typography, and dark/light mode behavior.

## Color Configuration

### Brand Colors

```json
"colors": {
  "primary": "#6366f1",
  "light": "#818cf8",
  "dark": "#4f46e5"
}
```

| Field | Description |
| --- | --- |
| `primary` | Base brand color. Applied to active navigation items, links, and accent elements. |
| `light` | Lighter shade. Used for hover states and accents in light mode. |
| `dark` | Darker shade. Used for hover states and accents in dark mode. |

### Extended Colors

For deeper customization, you can control the entire page palette:

```json
"colors": {
  "primary": "#F1BF4F",
  "light": "#F1BF4F",
  "dark": "#F1BF4F",
  "background": "#1a1a2e",
  "surface": "#242440",
  "surfaceElevated": "#2e2e4a",
  "foreground": "#f5f5f7",
  "mutedForeground": "#a0a0b0",
  "border": "rgba(255, 255, 255, 0.1)",
  "accent": "#60a5fa"
}
```

| Field | Description |
| --- | --- |
| `background` | Page background color. |
| `surface` | Header and sidebar background. |
| `surfaceElevated` | Elevated card/panel backgrounds. |
| `foreground` | Primary text color. |
| `mutedForeground` | Secondary/muted text color. |
| `border` | Border and divider color. |
| `accent` | Secondary accent color for interactive elements. |

## Theme Mode

Force your docs into dark or light mode regardless of the user's OS preference:

```json
"theme": {
  "mode": "dark",
  "radius": "0.5rem"
}
```

| Field | Description |
| --- | --- |
| `mode` | `"dark"`, `"light"`, or `"system"` (default). Forces a specific color scheme. |
| `radius` | Base border-radius value for UI elements. |

## Typography

Control heading, body, and code fonts independently:

```json
"fonts": {
  "family": "Inter Tight",
  "source": "https://fonts.gstatic.com/s/intertight/v7/...",
  "format": "woff2",
  "heading": {
    "family": "Fraunces",
    "source": "https://fonts.gstatic.com/s/fraunces/v31/...",
    "format": "woff2",
    "weight": "300 900"
  },
  "mono": {
    "family": "JetBrains Mono",
    "source": "https://fonts.gstatic.com/s/jetbrainsmono/v18/...",
    "format": "woff2"
  }
}
```

| Field | Description |
| --- | --- |
| `family` / `source` / `format` | Body font (applied globally). |
| `heading` | Optional separate font for h1-h4 headings. |
| `mono` | Optional separate font for code blocks and inline code. |

<Tip>
  Combine `theme.mode: "dark"` with custom `background` and `surface` colors to create a dark-themed documentation site that matches your product's look and feel.
</Tip>

## Dark Mode

Dory automatically follows the user's OS/browser dark mode preference. No configuration required — it works out of the box.

The `light` and `dark` color fields let you fine-tune how your brand color appears on light and dark backgrounds respectively.

<Tip>
  For best results: choose a `light` shade that's slightly lighter than `primary`, and a `dark` shade that's slightly darker. This ensures good contrast on both backgrounds.
</Tip>
