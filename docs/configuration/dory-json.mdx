# dory.json Reference

The `dory.json` file is the single configuration file for your Dory documentation site. It controls navigation, branding, analytics, and more.

## Full Example

```json
{
  "name": "My Docs",
  "title": "My Docs — Developer Reference",
  "description": "The official developer documentation.",
  "url": "https://docs.example.com",
  "image": "/og-card.png",
  "twitter": "@yourhandle",
  "favicon": "./favicon.svg",
  "colors": {
    "primary": "#6366f1",
    "light": "#818cf8",
    "dark": "#4f46e5",
    "background": "#1a1a2e",
    "surface": "#242440",
    "surfaceElevated": "#2e2e4a",
    "foreground": "#f5f5f7",
    "mutedForeground": "#a0a0b0",
    "border": "rgba(255, 255, 255, 0.1)",
    "accent": "#60a5fa"
  },
  "fonts": {
    "family": "Inter",
    "source": "https://fonts.gstatic.com/s/inter/v19/UcCO3FwrK3iLTeHuS_nVMrMxCp50SjIw2boKoduKmMEVuLyfAZ9hiJ-Ek-_EeA.woff2",
    "format": "woff2",
    "heading": {
      "family": "Fraunces",
      "source": "https://fonts.gstatic.com/s/fraunces/v31/...",
      "format": "woff2",
      "weight": "300 900"
    },
    "mono": {
      "family": "JetBrains Mono",
      "source": "https://fonts.gstatic.com/s/jetbrainsmono/v18/...",
      "format": "woff2"
    }
  },
  "theme": {
    "mode": "dark",
    "radius": "0.5rem"
  },
  "logo": {
    "light": "./logo/logo.svg",
    "dark": "./logo/logo-dark.svg"
  },
  "navigation": {
    "tabs": [
      {
        "tab": "Documentation",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["introduction", "installation"]
          }
        ]
      }
    ]
  },
  "analytics": {
    "posthog": {
      "token": "phc_your_token",
      "apiHost": "https://us.i.posthog.com"
    }
  },
  "embed": {
    "enabled": true,
    "defaults": {
      "position": "bottom-right",
      "width": 400,
      "height": 600,
      "theme": "system"
    }
  }
}
```

## Site Identity

| Field | Required | Description |
| --- | --- | --- |
| `name` | **Yes** | Site name shown in the header and browser tab. |
| `title` | No | Custom title override. Defaults to `name`. |
| `description` | No | Meta description for SEO and social sharing. |
| `url` | No | Canonical URL used when sharing links. |
| `image` | No | Preview image shown when links are shared on social media. |
| `twitter` | No | Your Twitter/X handle (e.g. `"@yourhandle"`). |
| `favicon` | **Yes** | Path to favicon file. Supports SVG and PNG. |

## Colors

```json
"colors": {
  "primary": "#6366f1",
  "light": "#818cf8",
  "dark": "#4f46e5"
}
```

| Field | Description |
| --- | --- |
| `primary` | Main brand color. Used for links, active states, and accents. |
| `light` | Lighter variant used in light mode. |
| `dark` | Darker variant used in dark mode. |

See [Theming](/configuration/theming) for more detail.

## Fonts

```json
"fonts": {
  "family": "Inter",
  "source": "https://fonts.gstatic.com/...",
  "format": "woff2"
}
```

| Field | Description |
| --- | --- |
| `family` | The font name (e.g. `"Inter"`, `"Montserrat"`). |
| `source` | Direct URL to the font file. |
| `format` | Font format — use `"woff2"` for best compatibility. |

See [Fonts](/configuration/fonts) for more detail.

## Theme

```json
"theme": {
  "mode": "dark",
  "radius": "0.5rem"
}
```

| Field | Description |
| --- | --- |
| `mode` | `"dark"`, `"light"`, or `"system"` (default). Forces the site into a specific color scheme. |
| `radius` | Base border-radius for UI elements. |

See [Theming](/configuration/theming) for full theming documentation.

## Logo

```json
"logo": {
  "light": "./logo/logo.svg",
  "dark": "./logo/logo-dark.svg"
}
```

| Field | Description |
| --- | --- |
| `light` | Path to logo for light mode. |
| `dark` | Path to logo for dark mode. Can be the same as `light` if your logo works on both backgrounds. |

```json
  "defaults": {
    "position": "bottom-right",
    "width": 400,
    "height": 600,
    "theme": "system",

    "closeOnEscape": true,
    "closeOnOverlayClick": true
  },
  "branding": {
    "showLogo": true,
    "showName": true
  },
  "features": {
    "search": true,
    "toc": true,
    "codeBlocks": true,
    "mermaid": true,
    "katex": true,
    "apiPlayground": true,
    "darkMode": true
  }
}
```

See [Embed](/configuration/embed) for embedding configuration.
