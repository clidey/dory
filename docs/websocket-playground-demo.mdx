## Testing Instructions

1. **Connect**: Click the "Connect" button to establish a WebSocket connection
2. **Send Messages**: Type messages in the input field and click "Send" or press Enter
3. **View Messages**: All sent and received messages appear in the message history with timestamps
4. **Configuration**: Use the tabs to configure headers, subprotocols, and authentication
5. **Server Selection**: Choose different servers from the dropdown to test various endpoints

## Features

- **Real-time Connection**: Live WebSocket connection with status indicators
- **Message History**: View all sent/received messages with timestamps
- **Custom Headers**: Add authentication and custom headers
- **Subprotocols**: Configure WebSocket subprotocols
- **Multiple Servers**: Easy switching between different WebSocket endpoints
- **Error Handling**: Clear error messages for connection issues
- **Binary Support**: Handles both text and binary messages

## Message Types

The playground supports different message types:
- **Sent**: Messages you send (blue background)
- **Received**: Messages from the server (green background)
- **Error**: Connection or sending errors (red background)
- **System**: Connection status updates (gray background)
